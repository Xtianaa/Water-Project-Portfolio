## Maji Ndogo Water Project - Performance Dashboard

![Maji Ndogo Water Project Performance Dashboard](Water_Project.PNG)

**Project Overview**

Developed a comprehensive performance dashboard to track the Maji Ndogo Water Project's progress in delivering clean water access across multiple provinces.

**Key Results:**
- Successfully completed **25,000 projects**
- Achieved **66%** basic water access for the target population
- Exceeded project budget with optimized resource allocation
- Identified key cost drivers (Public taps and Drill wells)

**Tools Used:** Power BI / Excel / SQL

**My Role:** Data cleaning, dashboard design, KPI development, and insight generation.
